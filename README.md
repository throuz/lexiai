# ⚖️ LexiAI — AI 法律文件摘要與問答平台

上傳法律 PDF → 自動文字擷取 → 使用 OpenAI 進行摘要與問答。

---

## 🔧 技術簡介

* **前端**：Next.js + Tailwind CSS
* **後端**：Flask + SQLAlchemy + OpenAI
* **資料庫**：PostgreSQL（Render 免費方案）
* **部署**：Render
